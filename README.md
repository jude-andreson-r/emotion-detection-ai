# Emotion Detection AI

## 📌 Overview
This project performs real-time emotion detection using a live camera feed.  
It detects human faces and classifies emotions such as happy, sad, angry, and neutral.

## 🛠 Technologies Used
- Python
- OpenCV
- DeepFace
- NumPy

## ✨ Features
- Real-time face detection
- Emotion classification using DeepFace
- Works with webcam input
- Simple and easy-to-run code

## ▶️ How to Run
1. Clone the repository
2. Install required libraries
3. Run the main Python file
4. Allow camera access

## 📂 Project Structure
emotion-detection-ai/
├── main.py
├── requirements.txt
└── README.md


## 📊 Output
The system displays detected emotions on faces in real time using the webcam.
